# 토스 버그바운티 테스트 체크리스트

## 공통 사전 점검
- [ ] User-Agent credential 고정 적용 완료
- [ ] 테스트 계정 A/B 준비 완료
- [ ] Burp CA 인증서/프록시 설정 완료
- [ ] 스코프 외 도메인 제외 완료
- [ ] 증거 저장 폴더(`evidence/YYYY-MM-DD/`) 생성

## 로그인/회원가입
- [ ] 로그인 요청/응답 캡처
- [ ] 실패 케이스 응답 캡처(오류 메시지 과노출 여부)
- [ ] 토큰 발급/갱신 요청 식별
- [ ] 확인 파라미터: `email`, `phone`, `otp`, `deviceId`, `session`
- [ ] 이상 징후: 계정 존재 유추 가능 메시지, 우회 로그인

## 계정 정보 수정
- [ ] 수정 전/후 조회 응답 캡처
- [ ] 타 계정 식별자 치환 단건 검증
- [ ] 확인 파라미터: `userId`, `nickname`, `phone`, `email`, `profileImageId`
- [ ] 이상 징후: 타 계정 정보 변경 성공, 권한 검증 부재

## 첨부파일 업로드/다운로드
- [ ] 업로드 요청/응답 캡처
- [ ] 파일 식별자 발급 방식 확인
- [ ] 다운로드 요청에서 파일ID 치환 단건 검증
- [ ] 확인 파라미터: `fileId`, `ownerId`, `contentType`, `path`, `signedUrl`
- [ ] 이상 징후: 권한 없는 다운로드, 만료 링크 재사용

## 외부 URL 입력 기능
- [ ] URL 입력/검증 요청 캡처
- [ ] `http://bugbounty.toss.sb`로만 SSRF 가능성 검증
- [ ] 확인 파라미터: `url`, `callback`, `previewUrl`, `target`
- [ ] 이상 징후: 서버측 임의 URL 요청, 필터 우회

## 결제/송금/금액 관련 요청
- [ ] 100원 기준 정상 요청 캡처
- [ ] 금액/수수료 파라미터 단건 변조 검증
- [ ] 확인 파라미터: `amount`, `fee`, `currency`, `recipientId`, `orderId`
- [ ] 이상 징후: 서버 재계산 부재, 클라이언트 금액 신뢰

## 고객센터/문의/알림/초대 기능
- [ ] 문의 등록/조회 요청 캡처
- [ ] 알림 읽음/삭제 요청 캡처
- [ ] 초대 링크 생성/검증 요청 캡처
- [ ] 확인 파라미터: `ticketId`, `message`, `inviteCode`, `notificationId`
- [ ] 이상 징후: 타 사용자 데이터 접근, 스크립트 입력 반사/저장

## 제출 전 최소 증거 세트
- [ ] 변조 전후 요청 1쌍
- [ ] 변조 전후 응답 1쌍
- [ ] 앱/웹 화면 캡처 1장 이상
- [ ] 타임스탬프 포함 로그
- [ ] 재현 절차 3단계 이상으로 정리
